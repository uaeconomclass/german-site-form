# Форма для німецького сайту

У цій папці ми готуємо/реалізовуємо веб-форму для німецького сайту на основі наданих документів (вимоги, текст, структура, поля, підказки тощо).

## Вихідні матеріали

Поточні файли-вимоги в корені папки:

- $pdf
- $docx

> Якщо зʼявляться нові версії документів, додаємо їх поруч і фіксуємо дату/версію в цьому README.

## Специфікація форми

Повний текстовий опис форми зі всіма полями, логікою, валідаціями та SMART-системою:
- **[FORM-SPEC.md](FORM-SPEC.md)** — головний документ специфікації

- [x] Підготувати тексти (лейбли, підказки, помилки) німецькою відповідно до документів.

## Нотатки

- Специфікація готова в `FORM-SPEC.md` — перед кодуванням варто її переглянути та узгодити.
- Мова інтерфейсу форми: німецька (згідно документів).
- Форма — багатокроковий wizard з 3 гілками (WG / NWG / Mischgebäude).
- SMART-система автоматично пропонує значення за роком будівництва та типом будівлі.
